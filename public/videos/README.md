# Hero Video Setup

To replace the hero video, please:

1. Place your MP4 video file in this directory
2. Name it `hero.mp4`
3. The video will automatically:
   - Autoplay on load
   - Loop continuously
   - Be muted (users can unmute if needed)
   - Display responsively as a background

## Video Specifications

- **Format:** MP4 (H.264 codec recommended)
- **Duration:** 15-30 seconds recommended
- **Resolution:** 1920x1080 (16:9 aspect ratio)
- **File Size:** Under 10MB for optimal loading

## Placement

Replace `/public/videos/hero.mp4` with your own video file.
